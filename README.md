Set the title of the [tmux](https://github.com/tmux/tmux) pane to the name of
the file you're editing in [vim](https://www.vim.org) or
[nvim](https://neovim.io/). If you're not using tmux the plugin doesn't do
anything. This plugin Just Works out of the box.

This is a standard Vim/Neovim plugin, please follow the instructions of your
plugin manager to install.
